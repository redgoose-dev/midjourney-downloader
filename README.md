# Midjourney Downloader

미드저니 이미지를 다운로드하는 크롬 확장 프로그램입니다.
상세 이미지 페이지에서 고정 다운로드 버튼을 표시하고, 버튼을 누르면 미드저니의 저장 메뉴를 열어 원본 이미지를 가로채 저장합니다.


## 동작 대상

- `https://www.midjourney.com/jobs/{CODE}` 형식의 페이지


## 기능

### 바구니 담기

지정한 다운로드 서브폴더 `~/Downloads/MJ-IMAGES` 위치로 이미지를 저장하고, 주요 텍스트 정보를 json 파일로 저장합니다.

### 이미지 저장

다운로드 폴더 `~/Downloads` 위치로 이미지를 저장합니다.


## 동작 방식

1. 페이지 하단에 `다운로드` 버튼이 표시됩니다.
2. 다운로드 버튼을 누르면 확장 프로그램이 대표 이미지를 찾습니다.
3. 이미지에 우클릭 이벤트를 보내 미드저니 컨텍스트 메뉴를 엽니다.
4. 컨텍스트 메뉴를 열고, 그 다음 `Save Image`를 선택합니다.
5. 다운로드 기능에 따라 지정한 위치로 저장하거거나 기본 다운로드 위치로 저장합니다.
6. 저작자, 프롬프트, URL 값들을 json 파일로 저장하기도 합니다.


## 개발 및 커스터마이즈

- `src/libs/assets.ts` 파일에서 주요 요소들을 수정하여 사용할 수 있습니다.
- 소스코드를 수정하면서 빌드하면 브라우저 확장프로그램에서 새로고침하거나 업데이트 버특을 꼭 눌러야합니다.


## 빌드

```
bun run build
```

빌드가 끝나면 `dist/` 경로에 js 파일과 manifest.json 파일이 만들어집니다.


## 설치 방법

1. 크롬에서 `chrome://extensions` 를 엽니다.
2. 우측 상단의 `개발자 모드` 를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다` 를 누릅니다.
4. `dist/` 폴더를 선택합니다.


## 참고

- 크롬 확장은 절대 경로를 직접 지정할 수 없어서 `~/Downloads/mj-images` 는 크롬 기본 다운로드 폴더가 `~/Downloads` 일 때만 정확히 일치합니다.
- 미드저니 UI 업데이트로 인하여 작동이 안될수도 있습니다.
